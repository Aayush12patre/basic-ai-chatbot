# 🤖 Basic AI Chatbot

A beginner-friendly AI chatbot web application built using Python, Streamlit, and Hugging Face Transformers. The chatbot accepts user input and generates AI-based text responses using the GPT-2 pretrained language model.

---

# 🚀 Features

- AI-powered text generation
- Simple and interactive web interface
- Real-time response generation
- Beginner-friendly implementation
- Public deployment using Streamlit Cloud

---

# 🛠️ Technologies Used

- Python
- Streamlit
- Hugging Face Transformers
- GPT-2
- PyTorch
